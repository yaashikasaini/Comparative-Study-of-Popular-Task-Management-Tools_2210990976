# Comparative Study of Popular Task Management Tools

## Overview

This research analyzes these task management tools based on:

- Usability
- Collaboration
- Customization
- Performance
- Cost effectiveness

## Methodology

- Testing period: 1 week
- Daily usage: 20–30 minutes
- Simulated academic project environment

## Key Findings

- Trello & Microsoft To Do → beginner friendly  
- Asana → strong team collaboration  
- ClickUp → advanced customization
